# CDS524 Assignment 1 - Q-Learning Grid World

## 📌 Project Overview
This project implements a reinforcement learning agent using the **Tabular Q-Learning** algorithm. The AI agent learns to navigate a 5x5 grid environment to find a hidden treasure (+100 reward) while avoiding deadly traps (-100 reward) and optimizing for the shortest path (-1 step penalty). 

## 🛠️ Technology Stack
* **Language:** Python
* **Algorithm:** Tabular Q-Learning ($\epsilon$-greedy policy, Bellman Equation)
* **Libraries:** NumPy (Q-table operations), Matplotlib & IPython.display (Dynamic UI rendering in Colab)
* **Environment:** Google Colab

## 🚀 How to Run
This project is fully optimized for Google Colab. No local installation of Pygame or other GUI libraries is required.
1. Open the `[Your_Colab_File_Name].ipynb` file in Google Colab.
2. Click **Runtime** -> **Run all**.
3. The first cell will train the agent for 500 episodes.
4. The second cell will output a dynamic, frame-by-frame emoji animation demonstrating the trained agent's optimal pathfinding.
